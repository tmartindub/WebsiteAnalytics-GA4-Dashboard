Glossary

GA4

Google Analytics 4.

GA4 Data API

The Google API used by this app to request report rows.

Property ID

The numeric GA4 property identifier used in API requests.

OAuth

Google sign-in used to authorize read-only analytics access.

Refresh token

A memory-only token that can obtain a new access token while the app keeps running.

Memory-only report data

Analytics rows are kept only while the app runs.
